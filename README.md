# Game--
####游戏 别热红球
#####游戏玩法
1.尽可能的闪开掉落的红球。

2.用边缘处接触红球 会有额外的加分。

3.掉落的白球每碰撞一次，就会有额外的分数加成，而且会变得 “更绿” 当然你的要能接住他才行。

4.你需要做取舍，当然也可以“玩火” 颠一下火球，艺高人胆大，有胆才会赢。


游戏特色

-测试你的敏捷和反应速度

-测试你的手脑并用

-有策略的取舍

-简单的控制

-锻炼手指灵活度

-看似简单,其实远没有你想象中的那么简单。

-3000分才算及格哦。



2.0新加入：

-4种飞碟，每一种飞碟都会掉落一种特殊的球。



-骷髅球 ：即死效果，躲它远远的。

-爆炸球：清空屏幕区域内所有的红球。
